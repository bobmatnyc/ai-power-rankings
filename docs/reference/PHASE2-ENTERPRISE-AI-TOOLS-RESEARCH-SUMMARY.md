## 1. JetBrains AI Assistant

### Company & Launch
- **Company**: JetBrains
- **Launch Year**: 2023
- **Website**: https://www.jetbrains.com/ai/

### Pricing Structure (2025)
- **AI Free**: $0/month (limited features, IDEs 2025.1+)
- **AI Pro Personal**: $8.33/month (yearly) or $10/month
- **AI Pro Commercial**: $16.67/month (yearly) or $20/month
- **AI Ultimate Personal**: $25/month (yearly) or $30/month
- **AI Ultimate Commercial**: $50/month (yearly) or $60/month
- **AI Enterprise**: Part of JetBrains IDE Services ($720/year base)

### Key Enterprise Differentiators
- **BYOLLM**: Bring your own LLM (OpenAI, Anthropic, Google, Azure)
- **On-premises deployment** via Hugging Face integration
- **Deep IDE integration** across entire JetBrains ecosystem
- **Credit-based pricing** for transparency ($1 = 1 credit)
- **SSO, audit logs, centralized policy management**

### Target Market
Enterprise development teams using JetBrains IDEs, organizations requiring on-premises AI, security-conscious enterprises needing BYOLLM flexibility

### Recent 2025 Updates
- Introduced simpler credit-based pricing (August 2025)
- Added AI Free tier for IDEs 2025.1+
- Enhanced Enterprise tier with BYOLLM and on-premises options
- Added Google Gemini via Vertex AI support

## 2. Amazon Q Developer

### Company & Launch
- **Company**: Amazon Web Services (AWS)
- **Parent**: Amazon
- **Launch Year**: 2024
- **Website**: https://aws.amazon.com/q/developer/

### Pricing Structure (2025)
- **Free**: $0/month (50 chat interactions, 10 agent invocations, 1,000 LOC transformation)
- **Pro**: $19/user/month (unlimited chat, unlimited agents, 4,000 LOC transformation)
- **Additional transformation**: $0.003 per line beyond quota

### Key Enterprise Differentiators
- **Autonomous AI agents** for multi-step development tasks
- **AWS deep integration** - native resource management
- **Legacy code transformation** (Java 8/11→17, .NET Framework→Core)
- **Built-in security scanning** and vulnerability detection
- **IAM Identity Center integration** for enterprise SSO
- **SOC/ISO/HIPAA/PCI compliance** eligibility
- **IP indemnification** for Pro tier

### Target Market
AWS cloud developers, enterprises migrating to AWS, DevOps teams, organizations in regulated industries requiring compliance

### Recent 2025 Updates
- Enhanced autonomous agent capabilities
- Expanded code transformation to .NET Framework migration
- Increased Pro tier transformation quota to 4,000 lines/month
- Added codebase customization for enterprises
- Improved security scanning with automated remediation

## 3. Google Gemini Code Assist

### Company & Launch
- **Company**: Google Cloud
- **Parent**: Google (Alphabet Inc.)
- **Launch Year**: 2024
- **Website**: https://cloud.google.com/gemini/docs/codeassist/overview

### Pricing Structure (2025)

- **Deep GCP integration** - BigQuery, Cloud Run, Apigee, Firebase
- **Gemini CLI** for command-line productivity
- **Agent mode** for autonomous multi-step tasks
- **BigQuery query generation** for data teams
- **Enterprise indemnification** and security

### Target Market
GCP developers, data scientists using BigQuery/Colab, API developers using Apigee, serverless teams on Cloud Run, enterprises needing GCP-integrated tools

### Recent 2025 Updates
- Launched Enterprise edition with private repository customization
- Introduced competitive $19/month Standard pricing
- Added promotional Enterprise pricing ($19/month until March 31, 2025)
- Enhanced code quality through organizational pattern learning
- Added Gemini CLI and improved Cloud Run integration

- **Custom AI code validation rules** and guardrails
- **Jira integration** for issue-driven development
- **24/5 enhanced support**

### Target Market
Enterprise teams with large multi-repository codebases, security-conscious organizations requiring self-hosted solutions, financial services, healthcare, companies needing BYOLLM flexibility

### Recent 2025 Updates
- Enhanced Advanced Code Graph for better multi-repo context
- Expanded BYOLLM support to Amazon Bedrock and Azure OpenAI
- Added customizable AI code validation rules
- Improved Jira integration for issue-driven development
- Introduced flexible token-based pricing for Enterprise

### Competitive Advantages
- Proven at scale (Uber, Lyft, Dropbox)
- Only tool with unlimited codebase connections
- Self-hosted and air-gapped options
- Superior Code Graph vs. competitors

## 5. Tabnine

### Company & Launch
- **Company**: Tabnine (formerly Codota)
- **Launch Year**: 2013
- **Website**: https://www.tabnine.com/

### Pricing Structure (2025)
- **Dev**: $9-12/user/month (basic features)
  - **Note**: Free Basic tier discontinued April 2025
- **Enterprise**: $39/user/month (full enterprise features)

### Key Enterprise Differentiators
- **Zero data retention** - code never stored on servers
- **Fully air-gapped deployment** - unique in market
- **On-premises Kubernetes** installation
- **Dell partnership** - turnkey GPU-accelerated air-gapped solutions (NVIDIA GTC 2025)
- **Tabnine Protected model** - trained only on permissively licensed code
- **GDPR compliance** (achieved 2025)
- **IP indemnification**
- **Advanced Context Engine** with unlimited codebase connections

### Target Market
Financial services, defense and government, healthcare, highly regulated enterprises, organizations requiring air-gapped environments, privacy-conscious teams

### Recent 2025 Updates
- Achieved GDPR compliance
- Discontinued free Basic tier (April 2025)
- Partnered with Dell for turnkey air-gapped GPU solutions
- Enhanced Code Review Agent with customizable validation
- Improved model flexibility with BYOM support

### Competitive Advantages
- **Only major AI assistant supporting fully air-gapped deployments**
- Industry-leading privacy with zero data retention
- Proven in highly regulated industries
- No vendor lock-in with BYOM flexibility

## 7. Windsurf

### Company & Launch
- **Company**: Codeium
- **Launch Year**: 2024 (late)
- **Website**: https://windsurf.com/

### Pricing Structure (2025)
- **Free**: $0/month (25 prompt credits = 100 premium model prompts)
  - **Increased from 5 credits** in early 2025
- **Pro**: $15/month (unlimited premium models)
- **Teams**: $30/user/month (team collaboration)
- **Enterprise**: $60/user/month (self-hosted, enhanced security)

### Key Enterprise Differentiators
- **AI-native IDE** - built from ground up for AI (not bolted on)
- **AI Flows paradigm** - combines Copilot + Agent capabilities
- **Cascade AI agent** - comprehensive codebase understanding
- **Self-hosted deployment** for enterprise security
- **40+ environment integrations**
- **In-editor live previews** for frontend development
- **Workflows** - automate tasks via natural language
- **Trusted by 1,000+ businesses** (Zillow, Dell, Anduril)

### Target Market
AI-first development teams, startups seeking velocity, enterprises requiring self-hosted AI (Fortune 500), frontend developers, DevOps teams automating workflows

### Recent 2025 Updates
- Increased free tier credits from 5 to 25 prompts/month
- Enhanced Cascade agent with deeper codebase understanding
- Expanded AI Flows capabilities
- Added GPT-4.1 and o4-mini model support
- Improved Workflows and live preview features
- Expanded enterprise integrations to 40+ environments

### Competitive Advantages
- **Only truly AI-native IDE** (vs. AI added to existing IDEs)
- Revolutionary Flows paradigm unique to Windsurf
- Generous free tier (25 credits = 100 premium prompts)
- Self-hosted enterprise deployment (rare among AI IDEs)
- Proven with Fortune 500 companies
